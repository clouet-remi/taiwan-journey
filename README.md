# Taiwan Journey — Site vitrine en Svelte 5

- Ce projet est un site vitrine développé avec Svelte 5. Je le réalise sur mon temps libre afin de m’entraîner aux technologies abordées dans ma formation et d’améliorer mes compétences en développement web. L’objectif est de proposer une interface claire destinée aux personnes qui souhaitent voyager, vivre ou étudier le chinois à Taïwan.

- Le site s’appuie sur plusieurs composants dédiés comme par exemple : un en-tête, une présentation des services, une section orientée vers les démarches d’étude et un pied de page. L’ensemble est conçu pour rester léger, lisible et facilement évolutif.
- Le projet est encore en cours et continuera de s’enrichir progressivement au fil de mon apprentissage : ajout de nouvelles sections, amélioration de la structure, travail sur le responsive et l’accessibilité, etc.

## Installation et lancement du projet

- Pour visualiser le site en local :

1.  `npm install`
2. `npm run dev`
